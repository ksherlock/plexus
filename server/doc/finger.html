Finger Gateway (w/faces)

There are two ways to use the Finger gateway. You can create hypertext links directly to http://server_host/finger?user@host.domain which is used to build a link that fingers someone (stored query) or you can use the gateway interactively using Search Keywords to search for user@host.domain. The cover page for the interactive finger is controlled by local.conf and should be customized for your site.

This code comes from the Marc VanHeyningen <mvanheyn@cs.indiana.edu> perl HTTP server whos finger gateway has more faces than mine.
