Directory Search

You can use the Directory Search function in search.pl when ever you have a small amount of data that you would like to allow full regular expression searches on. Just add a map for the base directory in the config file that points to &do;_search like the example in the distributed local.conf config file.

For example: 

map     info            search.pl       &do_search($path, $query, "Info")
This causes any document under /info that has the <ISINDEX> tag to allow a search of all the files in it's directory and all directories under it.
