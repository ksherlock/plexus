SpaceWalk Hypermedia Demo

NOTE: This demo isn't very interesting without at least NCSA Mosaic 1.1.

This demo allows you to select different regions of an image and have different actions take place (i.e., different documents are returned based on the X,Y coordinates of the mouse selection). You might also find it interesting to see what happens when you use a standard browser and select the image (you get a menu of the things you can select, this is for backwards compatibility).

The currently recognized regions are: outer space, earth, astronaut, camera and the space suit pack.

Hope you enjoyed this demo. For more information see the documentation and the source code.
